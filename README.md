# POC Vitrine Dynamique

## Objective

Python serverless API to fetch and enhance real estates classified ads, powered by AWS Rekognition for ads image tagging

## Infrastructure

- API Gateway
- Lambda
- DynamoDB (+ stream)
- Rekognition

## Software

- Python 3.6
- Flask
- Zappa
- PynamoDB

## Setup

```
cd vitrine-dynamique`
pipenv install
pipenv shell
./run-dev.sh
```
